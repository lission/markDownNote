# Algorithm

leetcode#206 [反转链表](https://leetcode.cn/problems/reverse-linked-list/)

迭代法

```java
    public ListNode reverseList(ListNode head) {
        //迭代法
        ListNode prev = null;
        ListNode curr = head;
        while(curr != null){
            ListNode next = curr.next;
            curr.next = prev;
            prev = curr;
            curr = next;
        }
        return prev;
    }
```
